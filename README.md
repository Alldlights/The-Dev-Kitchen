# The-Dev-Kitchen
A multi-section frontend blog project showcasing three distinct blog styles in one unified codebase. Built with vanilla HTML, CSS, and JavaScript, and enhanced with Bootstrap.

# Live Demo


## Sections
1. **Dev Journal** - Personal learning diary with dark mode, reading progress, and table of contents
2. **Recipe Blog** - Filterable recipe cards with dynamic modals and responsive images
3. **Startup Blog** - Professional marketing blog with newsletter signup and related posts carousel

## Technologies Used
- HTML5 (Semantic elements)
- CSS3 (Custom properties, Flexblox, Grid, BEM methodology)
- JavaScript(Components, Grid system, Utilities)
- JSON (Data Storage)

## Project Structure

# the-dev-kitchen/
1. index.html             (Homepage)
2. dev-journal.html       (Section 1)
3.  recipe-blog.html      (Section 2)
4. tartup-blog.html       (Section 3)


# /css/
- style.css                (Your custom styles - based on GitHub reference)
- variables.css            (Your CSS custom properties - colors, fonts)

# /js/
- main.js                  (Global functions - dark mode, navbar scroll)
- dev-journal.js           (Reading progress, TOC highlighting)
- recipe-blog.js           (Filtering, modals, smooth scroll)
- startup-blog.js          (Form validation, carousel, back-to-top)

# /assets/
- /images/                 (Placeholder images for recipes/startup)
- /icons/                  (SVG icons for social media, dark mode)
- /fonts/                  (Custom fonts if any)

# /data/
- posts.json               (ALL blog post data in one JSON file)

# README.md                (Project documentation)
